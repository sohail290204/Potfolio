Mad Wheels - Total Chaos

Mad Wheels Genre: Casual Driving Game
Engine: Unreal Engine 5.5
Platform: Android

Mad Wheels is a casual driving game where players must safely navigate through a chaotic city filled with reckless, unpredictable drivers. The game features five gameplay modes — Endless Run, Brake Fail, Speed Run (Timer), Parking, and Low Fuel — each offering a unique driving challenge. To keep gameplay fresh, players can choose from four different vehicles: Hatchback, Pickup, SUV, and Sports Car, each with its own strengths and weaknesses.

Development Details:

Engine: Built using Unreal Engine 5.5

Programming: Core game logic designed using a combination of Blueprints and C++ Vehicle System: Leveraged Unreal's Vehicle Plugin for physics-based driving mechanics

Assets:
Vehicle pack from Fab
Low-poly city buildings sourced from Sketchfab

UI & Visuals:
UI elements and menu buttons created with help from ChatGPT image generation

Audio:
Integrated MetaSounds for realistic vehicle sound Background music sourced from free online music libraries
